# Data

This repository does not redistribute raw data files.

The experiments use synthetic data generated directly in the notebooks, benchmark datasets obtained from public sources, and CPS data obtained through IPUMS CPS.

## Benchmark datasets

The benchmark experiments use:

- Engel data from statsmodels;
- Concrete Compressive Strength data from the UCI Machine Learning Repository;
- Abalone data from the UCI Machine Learning Repository;
- California Housing data from scikit-learn, after removing capped responses.

## CPS data

The large-scale CPS experiment uses data derived from the U.S. Current Population Survey, obtained through IPUMS CPS.

CPS data are not included in this repository. Users should obtain CPS data directly from IPUMS CPS and follow the provider's terms of use.
